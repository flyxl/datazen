## Description (260 chars)

DataZen manages PostgreSQL, MySQL, SQLite and Redis in one beautiful desktop app. Built with Rust for speed (under 10 MB). Features SQL editor, SSH tunnels, backups, PG-MySQL sync, dark mode and bilingual UI. MIT licensed.

## Maker comment (first comment)

Thanks for checking out DataZen! We built it because we wanted a TablePlus-like experience without the subscription — native speed via Tauri/Rust, credentials stay local (AES-256).

Early days (v0.0.3) but daily-driver ready for SQL + Redis. Would love feedback: wuxiaolongklws@gmail.com

macOS users: if Gatekeeper blocks launch, run `xattr -cr /Applications/DataZen.app` after install.
