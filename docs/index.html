DataZen

Lightweight, free, professional database client

轻量、免费、专业的桌面数据库客户端

Download GitHub Contact

Multi-database

PostgreSQL, MySQL, SQLite, and Redis in one native app (<10 MB).

SSH built-in

Connect through bastion hosts without a local ssh binary.

Local & secure

Credentials encrypted on disk (AES-256-GCM). No cloud account.

MIT licensed

Free and open source. macOS, Windows, and Linux.

macOS first launch — If macOS says the app is damaged, run after install: xattr -cr /Applications/DataZen.app 首次打开若提示「已损坏」，安装后执行上述命令即可。
