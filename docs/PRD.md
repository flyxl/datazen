## 1. 产品概述

### 1.1 产品定位

DataZen 是一款现代化的桌面数据库管理工具，类似于 TablePlus，采用多窗口架构设计，支持多种主流数据库，提供美观的用户界面和暗色模式支持。

### 1.2 目标用户

- 后端开发工程师
- 数据库管理员 (DBA)
- 数据分析师
- DevOps 工程师
- 全栈开发工程师

### 1.3 核心价值主张

1. **多窗口架构** - 每个数据库连接和查询独立窗口，互不干扰
2. **多数据库支持** - 统一界面管理 PostgreSQL、MySQL、SQLite、Redis 等
3. **现代美观界面** - 采用现代设计语言，支持暗色/亮色主题
4. **高性能** - 基于 Tauri + Rust，轻量快速
5. **安全可靠** - 本地存储，数据加密，无云端依赖

##### 交互说明

**CM-001 连接列表展示**
- 默认显示为卡片视图，可切换为列表视图
- 每张卡片显示：连接名称、数据库类型图标、主机地址、颜色标签、最后连接时间
- 卡片支持右键菜单：打开、编辑、复制、删除
- 双击卡片直接打开连接窗口
- 支持拖拽排序

**CM-002 新建连接**
- 点击"新建连接"按钮或使用快捷键 `Cmd+N`
- 弹出新建连接对话框
- 选择数据库类型后显示对应的配置表单
- 点击"测试连接"验证配置
- 点击"保存"保存连接配置

**CM-005 连接测试**
- 点击"测试连接"按钮
- 显示加载状态
- 成功：显示绿色成功提示，包含服务器版本信息
- 失败：显示红色错误提示，包含具体错误信息

## 3. 非功能需求

### 3.1 性能要求

| 指标 | 要求 |
|------|------|
| 启动时间 | < 2 秒 |
| 连接建立 | < 1 秒 |
| 查询响应 | 支持 10 万行结果流畅滚动 |
| 内存占用 | 空闲时 < 100MB |
| 安装包大小 | < 50MB |

### 3.2 安全要求

| 要求 | 描述 |
|------|------|
| 密码加密存储 | 使用 AES-256 加密存储数据库密码 |
| 本地存储 | 所有数据仅存储在本地，不上传云端 |
| SSH 隧道 | 支持通过 SSH 隧道连接数据库 |
| SSL 连接 | 支持通过 SSL 加密数据库连接 |

### 3.3 兼容性要求

| 类型 | 支持范围 |
|------|----------|
| 操作系统 | macOS 10.15+、Windows 10+、Linux (主流发行版) |
| 数据库版本 | PostgreSQL 12+、MySQL 5.7+、MariaDB 10.3+、SQLite 3.x、Redis 5+ |

### 3.4 可用性要求

| 要求 | 描述 |
|------|------|
| 错误提示 | 所有错误需有明确的中文错误提示 |
| 操作反馈 | 所有操作需有明确的视觉反馈 |
| 快捷键 | 所有常用操作支持快捷键 |
| 可恢复性 | 意外关闭后可恢复未保存的查询 |

## 5. 界面设计规范

### 5.1 设计原则

1. **简洁高效** - 减少不必要的元素，突出核心功能
2. **一致性** - 相同功能使用相同的交互方式
3. **响应式** - 及时反馈用户操作
4. **可访问性** - 支持键盘操作，清晰的视觉层次

## 6. 多窗口架构设计

### 6.1 窗口类型

| 窗口类型 | 描述 | 数量 |
|----------|------|------|
| 主窗口 | 连接管理、应用设置 | 单例 |
| 连接窗口 | 数据库浏览、数据编辑 | 每个连接一个 |
| 查询窗口 | SQL 编辑器 | 可创建多个 |

### 6.2 窗口管理

#### 主窗口行为
- 应用启动时显示
- 关闭时最小化到系统托盘（可配置）
- 负责管理所有连接窗口

#### 连接窗口行为
- 从主窗口双击连接打开
- 窗口标题格式：`{连接名称} - DataZen`
- 关闭时提示未保存的更改
- 支持从 Dock 图标快速访问最近连接

#### 查询窗口行为
- 可从连接窗口创建新的查询窗口
- 也可以在连接窗口内以标签页形式打开
- 窗口标题格式：`查询 - {数据库名} - DataZen`
- 关闭时提示未保存的查询

### 6.3 窗口间通信

```
主窗口
  │
  ├── 创建连接窗口 ──→ 连接窗口 1
  │                      │
  │                      ├── 打开数据表
  │                      ├── 打开查询编辑器
  │                      └── 发送查询到新窗口
  │
  └── 创建连接窗口 ──→ 连接窗口 2
                         │
                         └── ...
```

#### 通信方式
- 使用 Tauri 事件系统
- 全局状态使用文件存储
- 窗口间共享连接池

```
                    │

```
